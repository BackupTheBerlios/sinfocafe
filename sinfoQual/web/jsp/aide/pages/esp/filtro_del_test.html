Ayuda - Filtro del Test

 

                 Nombre

 

Descripción

Seleccionar el producto

Elegir el producto en la lista que aparece bajo el cartel.

Productor

Escribir el apellido del productor. Escribiendo el apellido, una lista aparece bajo el botón “confirmar”. Pues seleccionar el productor esperado y hacer un clic sobre el ligazón.

Grupo de interés

Escribir el grupo de interés del productor. Escribiendo el grupo de interés, una lista aparece bajo el botón “confirmar”. Pues seleccionar el grupo de interés esperado y hacer un clic sobre el ligazón.

Asociación

Elegir una asociación entre las cinco propuestas.

Tipo de café

Elegir un tipo de café entre las diferentes propuestas.

Lote

Escribir el número de lote.

Fecha

Elegir la fecha del test.

Confirmar

Apretar este botón cuando ha terminado de llenar las informaciones necesarias.
