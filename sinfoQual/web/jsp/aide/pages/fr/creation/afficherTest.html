Aide - Affichage du test

 

Une fois le test retrouvé parmi les archives, vous pouvez

effectuer différentes actions: imprimer, générer un

document, ou encore envoyer ces résultats par email.

 

Intitulé

 

Description

Imprimer

Imprimer la page résumant tous les résultats du test de café recherchés.

Générer un document

Générer un document contenant les résultats du test recherchés précédemment.

Envoyer par mail les résultats

Envoyer un courrier électronique avec un document attaché contenant les résultats du test recherchés précédemment.
