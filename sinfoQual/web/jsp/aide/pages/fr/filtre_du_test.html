Aide - Filtre du test

 

                 Intitulé

 

Description

Sélectionner le produit

Choisir le produit dans la liste déroulante.

Producteur

Entrer le nom du producteur. En entrant le nom, une liste apparaît sous le bouton "Confirmer". Sélectionner ensuite le producteur voulu et cliquer sur le lien.

Groupe d'intérêt

Entrer le nom du groupe d'intérêt. En entrant le nom, une liste apparaît sous le bouton "Confirmer". Sélectionner ensuite le groupe d'intérêt voulu et cliquer sur le lien.

Association

Choisir une association parmi les cinq proposées.

Type de café

Choisir un type de café parmi les propositions.

Lot

Ecrire le numéro du lot.

Date

Choisir la date du test.

Confirmer

Presser le bouton lorsque vous avez fini d'entrer toutes les informations que vous voulez.
